Pares de puntos más cercanos

Diviertete aprendiendo a resolver el problema de "pares de puntos más cercanos" con una animación interactiva.

¡Vamos!

  1. Da click en cualquier parte dentro del marca para dibujar un punto, o puedes presionar puntos aleatorios ( )
  2. Existe un limite de hasta 20 puntos
  3. Presiona "iniciar animación" para ver la solución por fuerza bruta ( )
  4. Presiona "Divide & Conquer" para ver la solución optimizada ( )
  5. Presiona "Limpiar" para reiniciar la animacion ( )
